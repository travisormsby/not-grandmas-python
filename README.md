Materials for MN GIS/LIS 2026 Spring Workshop

Follow these instructions to get started:

1. Click **Code**, then click the **Codespaces** tab.
1. Click **Create a codespace on main**

    The repository will open with a codespace instance of VS Code. There will be a bash terminal open at the bottom.

1. Open the workshop notebook `this-aint-your-grandmas-python.ipynb`.

1. In the upper right, click **Select Kernel**

1. From the dropdown, select the option to install recommended extensions Python + Jupyter
